# Functions

In this folder, you will learn how to write clean, modular, and reusable code using Python functions.

## Concepts covered:
- **Defining Functions**: Using the `def` keyword to create your own tools.
- **Parameters and Arguments**: Passing data into functions to make them flexible.
- **Return Statements**: Sending data back out of a function so it can be used elsewhere in your program.
- **Scope**: Understanding the difference between global and local variables.
- **Default Parameters**: Setting values that are used if the caller doesn't provide them.
- **Docstrings**: Writing documentation inside your functions so others (and your future self) understand how they work.
