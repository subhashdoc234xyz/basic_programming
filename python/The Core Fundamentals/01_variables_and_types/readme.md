# Variables and Data Types

In this folder, you will learn how Python stores and manages data.

## Concepts covered:
- **Variables**: Assigning data to names using the assignment operator (`=`).
- **Data Types**:
    - `int`: Whole numbers (e.g., 5, -10).
    - `float`: Numbers with decimal points (e.g., 3.14, -0.01).
    - `str`: Text data enclosed in quotes (e.g., "Hello World").
    - `bool`: Truth values (`True` or `False`).
- **Type Conversion**: Changing types using functions like `int()`, `float()`, and `str()`.
- **Dynamic Typing**: Understanding that Python determines the variable type at runtime.
