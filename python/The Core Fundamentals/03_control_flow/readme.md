

# Control Flow

In this folder, you will learn how to make your programs "think" by using conditional logic.

## Concepts covered:
- **`if` statements**: Executing code only when a condition is `True`.
- **`elif` statements**: Adding alternative conditions.
- **`else` statements**: Catching cases where no previous conditions are met.
- **Nested Conditionals**: Placing `if` statements inside other `if` statements.
- **Truthiness**: Understanding which values in Python are treated as `True` or `False` (e.g., empty strings or 0 are `False`).
