# Loops

In this folder, you will learn how to repeat blocks of code efficiently.

## Concepts covered:
- **`for` loops**: Iterating over sequences (like a range of numbers or items in a list).
- **`while` loops**: Repeating code as long as a specific condition remains `True`.
- **Loop Control Statements**:
    - `break`: Exiting a loop entirely.
    - `continue`: Skipping the rest of the current iteration and moving to the next one.
- **Infinite Loops**: How to recognize them and how to avoid them in `while` loops.
