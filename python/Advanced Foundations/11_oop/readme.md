# Object-Oriented Programming (OOP)

In this folder, you will learn how to model real-world concepts into software using Classes and Objects.

## Concepts covered:
- **Classes and Objects**: Blueprints for creating objects (`class`, `__init__`, `self`).
- **Methods**: Functions that belong to a class.
- **Inheritance**: Creating child classes that inherit attributes and methods from parent classes.
- **Encapsulation**: Restricting direct access to some components (using `_` or `__` prefixes).
- **Polymorphism**: The ability for different classes to respond to the same method call in their own way.
