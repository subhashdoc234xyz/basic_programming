# File Handling

In this folder, you will learn how to interact with the file system to persist data beyond the life of your program.

## Concepts covered:
- **Opening Files**: Using the `open()` function with different modes (`'r'` for read, `'w'` for write, `'a'` for append).
- **The `with` Statement**: Using context managers to ensure files are closed automatically, even if errors occur.
- **Reading Data**: Techniques for reading lines or the entire file content (`read()`, `readline()`, `readlines()`).
- **CSV Handling**: Introduction to the built-in `csv` module for structured data management.
