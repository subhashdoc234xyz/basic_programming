# Dictionaries

In this folder, you will learn how to store data in key-value pairs, which is essential for mapping relationships.

## Concepts covered:
- **Definition**: Unordered (pre-3.7) or insertion-ordered (3.7+) collection of key-value pairs (defined with `{}`).
- **Accessing Data**: Using keys to retrieve values efficiently.
- **Modification**: Adding, updating, and deleting key-value pairs.
- **Methods**: Iterating through `keys()`, `values()`, and `items()`.
- **Use Cases**: Best practices for storing structured data that needs quick lookup.
