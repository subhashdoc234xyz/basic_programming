# Lists

In this folder, you will learn about Python lists, which are the most common way to store a sequence of items.

## Concepts covered:
- **Definition**: Ordered, mutable collections (defined with `[]`).
- **Indexing**: Accessing items by their position (starting at 0).
- **Slicing**: Extracting sub-sections of a list.
- **Modification**: Adding (`append`, `insert`), removing (`pop`, `remove`), and changing items.
- **Methods**: Useful built-in functions like `sort()`, `reverse()`, and `len()`.
