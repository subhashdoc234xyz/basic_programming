# Object-Oriented Programming (OOP)

In this folder, you will learn the core pillars of Java's design.

## Concepts covered:
- **Classes and Objects**: Defining blueprints and instantiating objects.
- **Constructors**: Initializing objects.
- **Encapsulation**: Using `private` fields and `public` getters/setters.
- **Inheritance**: Using the `extends` keyword.
- **Polymorphism**: Overriding methods and using interfaces.
